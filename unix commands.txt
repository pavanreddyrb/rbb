ls: List files and directories in the current directory.
cd: Change directory.
pwd: Print the current working directory.
mkdir: Create a new directory.
rm: Remove files and directories.
cp: Copy files and directories.
mv: Move or rename files and directories.
cat: Concatenate and display the contents of files.
less: View the contents of a file one page at a time.
head: Display the first few lines of a file.
tail: Display the last few lines of a file.
grep: Search for a pattern in files.
chmod: Change the permissions of files and directories.
chown: Change the ownership of files and directories.
ssh: Connect to a remote server via SSH.
scp: Copy files between local and remote systems using SSH.
wget: Download files from the web.
tar: Create or extract tar archives.
gzip: Compress files.
gunzip: Decompress files.
man: Display the manual page for a command.
history: View the command history.
top: Display system resource usage and running processes.
ps: Display running processes.
kill: Terminate a process.
touch: to create the file.
